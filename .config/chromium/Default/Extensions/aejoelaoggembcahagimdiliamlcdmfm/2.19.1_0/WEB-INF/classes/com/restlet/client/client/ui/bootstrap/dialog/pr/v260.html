July 19, 2017 - Restlet Client 2.6.0

Dive deep into your JSON response

The Expression Builder has been designed to help you take the better out of expressions , like reusing data from previous requests and responses.

However until now extracting an information from a JSON response was not really easy. We worked on it, so that the Expression Builder now lets you dive into JSON content and pick a particular property without having to type a single character.

July 19, 2017 - Restlet Client 2.6.0

How to query the query?

Thanks to your feedback we noticed that we did not provide an easy way to extract a query parameter to use it in either an assertion or in a subsequent request.

Extracting a query parameter is now as simple as an expressions on a request like: 

${"My Project"."My Scenario"."Request 1"."request"."query"."my query parameter name"}

Or simply use the Expression Builder and select the query parameter you like.

July 19, 2017 - Restlet Client 2.6.0

New project and service overviews

Project and service pages have been redesigned to offer better information presentation and let you do more directly from these pages. From now on, renaming and editing the children can be done immediately from the overview which let you document all your data from one single place.

Additionally for Scenarios view users, it is now possible to run the content of a whole project (or a service) in a single click!
