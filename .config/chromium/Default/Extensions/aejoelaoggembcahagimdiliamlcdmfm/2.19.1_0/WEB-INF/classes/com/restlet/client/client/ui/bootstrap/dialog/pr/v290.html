September 06, 2017 - Restlet Client 2.9.0

Validate your API's behavior, it’s easy! ✨

In this version, we’ve made assertions easier to approach and to use.

  • Assertion labels have been improved for more clarity
  • Setting assertions on headers was made simpler by adding completion on header names
  • A new documentation page is available to learn more about how to get a specific part of a body thanks to JSONPath or XPath.
  • Last but not least, Restlet Client now suggests assertions to add based on current request and response.

Assertions matter, so if you have comments to share, we'd be thrilled to hear about it. You can reach out to us via in-app chat or support@restlet.com.
