November 06, 2017 - Restlet Client 2.11.0

New assertions operators including regex!

Based on your feedback we’ve added more expressivity in our assertion engine. Check out the improvements we've made:

  • Matches allows you to validate any value against a pattern defined by a JavaScript regular expression
  • Length lets you check the length of an item. It checks the number of characters of strings, the number of items in a JSON array and the number of keys in a JSON object - just like the expression's length operator!
  • Contains is now smarter and lets you test that a set of value is included in a JSON array. However please note these changes may break some of your existing “contains” assertions.
